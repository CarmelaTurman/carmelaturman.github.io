CSS Media Queries

How does a CSS media query work, and why is it useful?

A CSS media query is a technique used to apply CSS styles conditionally, based on certain features of the user’s device or browser. Media queries allow developers to create responsive designs that adapt to different screen sizes, resolutions, orientations, or other properties. This makes websites work well on desktops, tablets, and smartphones without requiring separate code for each device.

Supported Media Features and Examples

Here are four common media features and their uses:

  • max-width: Applies styles only if the viewport width is less than or equal to a specified value. Example: @media (max-width: 600px) will apply styles for screens narrower than 600px, such as smartphones.
  • min-width: Applies styles only if the viewport width is greater than or equal to a specified value. Example: @media (min-width: 1024px) targets larger devices like desktops.
  • orientation: Targets the orientation of the device, either portrait or landscape. Example: @media (orientation: portrait) applies styles when the device is held vertically.
  • prefers-color-scheme: Allows developers to detect if the user prefers a light or dark color theme. Example: @media (prefers-color-scheme: dark) applies styles for dark mode.
