Code Log: CSS Color

Topic: How CSS uses color in design

How CSS Uses Color

CSS lets us add color to almost anything on a web page — text, backgrounds, borders, and more. Using color correctly can make a site look professional, readable, and visually balanced. Here’s how some key color-related properties work:

  • color — changes the text color inside an element.
  • background-color — fills the background of an element with a chosen color.
  • border-color — sets the color of the border around an element (you’ll only see it if a border is defined).

Ways to Set Color Values

CSS can describe colors in several ways. Here are four common formats:

1. Named Colors

CSS includes over 140 named colors like red, blue, tomato, or lightgray.

Example: background-color: tomato;

2. Hexadecimal Colors

Hex codes use numbers and letters to describe colors, starting with a #. For example, #1e90ff represents a bright blue. Each pair of characters represents the amount of red, green, and blue (RRGGBB).

Example: background-color: #1e90ff;

3. RGB Colors

The rgb() function lets you specify exact amounts of red, green, and blue as numbers between 0 and 255. You can also use rgba() to add transparency (the “a” stands for alpha).

Example: background-color: rgb(60, 179, 113);

4. HSL Colors

hsl() stands for hue, saturation, and lightness. It’s easier for humans to tweak, since you can think in terms of how intense or bright a color is.

Example: background-color: hsl(300, 50%, 40%);

Summary

CSS color options are flexible and powerful. You can express the same color in multiple ways, depending on what’s easiest for your workflow. Whether you use named colors for simplicity, hex codes for precision, or RGB/HSL for control — CSS gives you all the tools to make your website visually stand out.
